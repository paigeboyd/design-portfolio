Css illustrations

This is a collection of little illustrations I have created using code. I work with HTML and CSS to create various shapes then concentrate on the positioning to bring them together to create the final illustration. This is one of the few times where my process doesn't begin on paper!

This is side project was started just to allow me to practice coding but bring a creative twist to it. I keep them on CodePen.


My Process

During the summer between Year 2 and Year 3, I wanted to get better at CSS, so I began watching some YouTube tutorials when I came across the method of creating illustrations using CSS. The process is simple in itself; breaking an illustration down to basic shapes, then creating divs which are styled and positioned to create the illustration. Some illustrations can be quite complex and so deciding how to create the individual shapes and how to position them can be quite challenging, but it is rewarding once the illustration finally takes shape!
